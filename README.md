# Лабораторна робота №7

Тема: Використання реактивних інструментів управління потоком даних.

Проєкт реалізує Todo-список з використанням RxJS.

## Що реалізовано

- збереження стану списку через BehaviorSubject;
- обробка подій користувача через fromEvent;
- використання pipe, map, filter;
- об'єднання потоку даних та стану інтерфейсу через combineLatest;
- додавання, редагування, видалення і фільтрація завдань.
