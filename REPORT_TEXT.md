У ході виконання лабораторної роботи було створено інтерактивну сторінку Todo-списку з використанням бібліотеки RxJS. Стан списку завдань зберігається у реактивному контейнері BehaviorSubject, що дозволяє централізовано оновлювати дані та автоматично передавати зміни в інтерфейс.

Дії користувача обробляються через функцію fromEvent. Подія відправлення форми використовується для додавання нового завдання, подія зміни select — для перемикання фільтра, а кліки по списку — для редагування, видалення та зміни статусу завдання. Для обробки потоків застосовано pipe, оператор map і фільтрацію значень.

Для автоматичного оновлення інтерфейсу використано combineLatest, який поєднує потік даних списку та потік стану фільтра. Коли змінюється список або обраний фільтр, формується нова модель відображення, після чого HTML-елементи на сторінці перемальовуються автоматично.

Висновок: під час виконання лабораторної роботи було розглянуто реактивний підхід до керування потоками даних у вебінтерфейсі. Було реалізовано збереження стану в BehaviorSubject, обробку DOM-подій через fromEvent, перетворення подій за допомогою pipe та map, а також автоматичне оновлення сторінки через combineLatest.
